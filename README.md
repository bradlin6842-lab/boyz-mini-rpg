# Boyz Mini RPG

使用 **Phaser 3 + Vite + TypeScript** 建立的 2D 小遊戲骨架專案。

## 啟動方式

```bash
npm install
npm run dev
```

## 操作方式

啟動後可使用下列方式移動角色：

- 鍵盤方向鍵（↑ ↓ ← →）
- WASD（W A S D）
- 螢幕右下角虛擬方向鍵（支援手機 / iPad 觸控點擊與長按）

## 目前測試內容（戰國小村莊 v1）

目前地圖為 **20x15 tile-based** 的「戰國小村莊」測試圖，已可驗證：

- 基礎移動（方向鍵 / WASD / 虛擬方向鍵）
- 地圖元素顯示（草地、草叢、石板路、神社、道館、NPC、木牌/石燈籠）
- 地圖碰撞
  - 可通行：草地、石板路、草叢
  - 不可通行：神社、道館牆面、木牌、石燈籠、地圖邊界

> 目前尚未加入野外遇敵、戰鬥、捕捉、道具功能。

## 資料夾結構

- `src/scenes`：場景
- `src/entities`：角色/物件
- `src/systems`：遊戲系統（例如移動）
- `src/data`：地圖或靜態資料
- `public/assets`：靜態資源
