# Boyz Mini RPG

使用 **Phaser 3 + Vite + TypeScript** 建立的 2D 小遊戲骨架專案。

## 啟動方式

```bash
npm install
npm run dev
```

## 操作方式

啟動後可使用下列方式移動角色：

- 鍵盤方向鍵（↑ ↓ ← →）
- WASD（W A S D）
- 螢幕右下角虛擬方向鍵（支援手機 / iPad 觸控點擊與長按）

互動可使用：

- 鍵盤 `E`
- 畫面左下方半透明「互動」按鈕（適用 iPad / 觸控）

## 目前測試內容（戰國小村莊 v4）

目前地圖為 **20x15 tile-based** 的「戰國小村莊」測試圖，已可驗證：

- 基礎移動（方向鍵 / WASD / 虛擬方向鍵）
- 玩家主角已升級為像素風「戰國少年冒險者」sprite
  - 四方向（上 / 下 / 左 / 右）
  - 每方向 1 張 idle + 2 張 walk
  - 移動時會播放 walking animation，停止後回到對應方向 idle
- 地圖元素顯示（草地、草叢、石板路、神社、道館、NPC、木牌/石燈籠）
- 地圖碰撞
  - 可通行：草地、石板路、草叢
  - 不可通行：神社、道館牆面、木牌、石燈籠、地圖邊界
- NPC 與地圖互動提示（靠近 NPC / 神社 / 道館入口 / 木牌會顯示「按 E 互動」）
- 對話框互動（黑底白字）
  - 村民 NPC：歡迎來到 Boyz 村，草叢裡聽說有妖怪出沒。
  - 木牌：北方是神社，東方是道館。
  - 神社：神社散發溫暖的光，目前補血功能尚未開放。
  - 道館入口：道館大門緊閉，似乎需要先獲得第一隻妖怪。
- 對話期間玩家無法移動，按 `E` / 互動按鈕 / 點擊對話框可關閉
- 對話框已加入主角 portrait（可供後續對話 UI 擴充）
- 草叢遇敵（走進深綠色草叢 tile 時有 8% 機率觸發）
  - 野生妖怪：狸貓妖 / 火狐 / 石燈籠
- 第一版可玩 BattleScene（回合制）
  - 玩家初始妖怪為「啾啾」（HP 35 / Attack 9 / Defense 5）
  - 戰鬥 UI 顯示：玩家與野生妖怪名稱、雙方 HP、剩餘封印符、戰鬥訊息
  - 攻擊可造成傷害，傷害公式：`max(1, attack - defense + random(0~4))`
  - 野生妖怪未倒下時會立即反擊
  - 打倒野生妖怪後可按「確認」返回村莊地圖
  - 玩家 HP 歸零會顯示提示並送回村莊，啾啾 HP 自動回滿
  - 捕捉會消耗 1 張封印符（起始 5 張），符紙用盡時不消耗回合
  - 捕捉成功率會依野怪種類 baseCatchRate 與目前血量計算，血量越低越容易捕捉
  - 捕捉成功後按確認返回村莊地圖；失敗則野怪反擊
  - 保留「逃跑」按鈕可直接返回村莊地圖
  - 所有戰鬥按鈕（攻擊 / 捕捉 / 逃跑 / 確認）支援滑鼠與觸控（iPad）
  - 妖怪圖像目前為 pixel placeholder

> 目前尚未加入升級、進化、道館 Boss、圖鑑或背包功能。

## 資料夾結構

- `src/scenes`：場景
- `src/entities`：角色/物件
- `src/systems`：遊戲系統（例如移動）
- `src/data`：地圖或靜態資料
- `public/assets`：靜態資源
