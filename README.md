# Boyz Mini RPG

使用 **Phaser 3 + Vite + TypeScript** 建立的 2D 小遊戲骨架專案。

## 啟動方式

```bash
npm install
npm run dev
```

## 操作方式

啟動後可使用下列方式移動角色：

- 鍵盤方向鍵（↑ ↓ ← →）
- WASD（W A S D）
- 螢幕右下角虛擬方向鍵（支援手機 / iPad 觸控點擊與長按）

互動可使用：

- 鍵盤 `E`
- 畫面左下方半透明「互動」按鈕（適用 iPad / 觸控）

## 目前測試內容（戰國小村莊 v2）

目前地圖為 **20x15 tile-based** 的「戰國小村莊」測試圖，已可驗證：

- 基礎移動（方向鍵 / WASD / 虛擬方向鍵）
- 地圖元素顯示（草地、草叢、石板路、神社、道館、NPC、木牌/石燈籠）
- 地圖碰撞
  - 可通行：草地、石板路、草叢
  - 不可通行：神社、道館牆面、木牌、石燈籠、地圖邊界
- NPC 與地圖互動提示（靠近 NPC / 神社 / 道館入口 / 木牌會顯示「按 E 互動」）
- 對話框互動（黑底白字）
  - 村民 NPC：歡迎來到 Boyz 村，草叢裡聽說有妖怪出沒。
  - 木牌：北方是神社，東方是道館。
  - 神社：神社散發溫暖的光，目前補血功能尚未開放。
  - 道館入口：道館大門緊閉，似乎需要先獲得第一隻妖怪。
- 對話期間玩家無法移動，按 `E` / 互動按鈕 / 點擊對話框可關閉

> 目前尚未加入野外遇敵、戰鬥、捕捉、道具功能。

## 資料夾結構

- `src/scenes`：場景
- `src/entities`：角色/物件
- `src/systems`：遊戲系統（例如移動）
- `src/data`：地圖或靜態資料
- `public/assets`：靜態資源
