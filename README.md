# Boyz Mini RPG

使用 **Phaser 3 + Vite + TypeScript** 建立的 2D 小遊戲骨架專案。

## 啟動方式

```bash
npm install
npm run dev
```

## 操作方式

啟動後可使用下列方式移動角色：

- 鍵盤方向鍵（↑ ↓ ← →）
- WASD（W A S D）
- 螢幕右下角虛擬方向鍵（支援手機 / iPad 觸控點擊與長按）

## 資料夾結構

- `src/scenes`：場景
- `src/entities`：角色/物件
- `src/systems`：遊戲系統（例如移動）
- `src/data`：地圖或靜態資料
- `public/assets`：靜態資源
